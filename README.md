# Mahesh Merchants Website

A highly polished, responsive, and bilingual landing page for **Mahesh Merchants**, a grocery and provisions store located near Bhulokamma Temple & Eluru Railway Station in Eluru, Andhra Pradesh.

## Features
* **Bilingual Support (English & తెలుగు):** Dynamic translation toggle built directly into the header menu.
* **Local SEO Optimized:** Pre-integrated with schema markup (`LocalBusiness` JSON-LD) for better Google Maps discoverability in Eluru.
* **Interactive Product Catalog:** Category filtering (All, Grains, Spices, Essentials) with smooth transitions.
* **WhatsApp Order Enquiries:** Direct WhatsApp chat integration containing pre-templated messages matching the user's selected language.

## Getting Started
To preview and run this project locally:

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the Vite development server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173) in your browser.
